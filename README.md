# 🚀 AI Mentor for JEE/NEET

An intelligent **AI-powered mentor system** designed for students preparing for **JEE / NEET**.  
It helps students solve doubts, track progress, and identify weak subjects — just like a real mentor.

---

## ✨ Features

### 🧠 1. Doubt Solving System
- Ask doubts in **Physics, Chemistry, Maths, Biology**
- 3 smart modes:
- ⚡ Quick Doubt Mode → short answers
- 🎯 Exam Mode → strategy + important topics
- 📘 Depth Mode → detailed explanation

---

### 👤 2. Student Profile
- Store student data:
- Name
- Current Class
- Target Exam
- Weak Subject

---

### 📊 3. Weak Subject Analysis
- Automatically detects weak subject based on:
- Number of doubts asked per subject
- Helps students focus on improvement areas

---

### 📈 4. Progress Tracking
- Tracks:
- Total doubts asked
- Subject-wise distribution
- Gives a clear learning overview

---

### 📜 5. Doubt History
- Stores all previous doubts
- Allows students to review past learning

---

## 🛠️ Tech Stack

- **Backend:** FastAPI
- **AI Model:** Google Gemini API
- **Frontend:** HTML, CSS, JavaScript
- **Language:** Python
